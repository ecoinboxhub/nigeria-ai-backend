 Codex: Nigeria Construction AI Platform - Project Overview
The Codex project is a sophisticated AI-powered construction management backend designed for Nigerian infrastructure projects. It aims to optimize various construction aspects using a modular, data-driven approach.

🏗️ Mission and Scope
Domain: AI-driven construction management.
Focus Areas: Delay risk scoring, procurement intelligence, safety analysis, cost forecasting, workforce optimization, and more.
MVP Status: Backend-only service with synthetic data bootstrapping for rapid development and evaluation.
🛠️ Technical Stack
API Framework: FastAPI (modular architecture with RBAC and JWT authentication).
ML Operations:
DVC: Data Version Control for reproducible data and model pipelines.
MLflow: Experiment tracking and model management.
Infrastructure: Docker & Docker Compose for local development; Render for cloud deployment.
Databases: PostgreSQL (transactional), MongoDB (semi-structured logs), Chroma/Pinecone (vector storage for RAG).

### 1. How to Run the Backend Locally

The backend is built with Python and FastAPI. It uses a virtual environment to manage dependencies so they don't conflict with your global system installs.

**Step-by-Step:**
1. Open a new PowerShell or Command Prompt terminal.
2. Navigate to the backend directory:
   ```powershell
   cd C:\Users\ibrah\Documents\AntiGravity\nigeria-ai-backend
   ```
3. Activate the virtual environment:
   ```powershell
   .\.venv\Scripts\Activate.ps1
   ```
   *(You should see [(.venv)](cci:1://file:///c:/Users/ibrah/Documents/AntiGravity/nigeria-ai-backend/backend/app/db/session.py:14:0-19:18) prefix your terminal prompt now, indicating it's active).*
4. Navigate into the `backend` folder where the source code lives:
   ```powershell
   cd backend
   ```
5. Start the FastAPI server using Uvicorn:
   ```powershell
   python -m uvicorn app.main:app --reload --port 8000
   ```
6. **Success!** The backend is now running. You can view the automatically generated interactive API documentation by visiting [http://127.0.0.1:8000/docs](http://127.0.0.1:8000/docs) in your browser.
